How to understand the Grid

Understanding CSS is not about learning every property and value by heart. Core ideas that help CSS layout makes sense. You have to understand it to really utilize the Grid

  1. Understanding Cascading Style Sheets
  2. Inheritance - which properties will inherit values from their parent.
  3. Specificity - which rule wins when two things could apply to the same element.
  4. Layout

You can fake a grid with flexbox and its very simlar as before, with math and percentages but past layout methods create the appearance of a grid, by lining things up.

CSS Intrinsic and Extrinsic Sizing

There are different ways to set width: by pixel size or container meaning the text or content dictates the size. Items start by trying to display at max-content size.

Flex lets you choose the items sizes and dictate how they behave. flex: 1 1 auto; shares the space of a container and spaces them out equally

flex: 1 1 0;With a flex-basis of 0 space is distributed from 0, making equal columns.

Flexbox is starting from max-content and taking space away. Grid starting at min-content and adding space.

In grid there is a value called fit-content and you can add value to it
