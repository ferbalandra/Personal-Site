What is a progressive web app

A progresive web App is a website that has been enhanced with securtiy, service worker {manifest}

Service worker is the key for web apps explicit control of network req uest and cache. Service workers are also key providing an ofline experience and push notifications

Progressive web apps allow us to build experiences that previously required a native application.

“Mobile web audiences are almost 3x the size and fast as app audiences.” Your website is often the first interaction with your company. A better web experience will likely mean more revenuew and more native app onstalls.
